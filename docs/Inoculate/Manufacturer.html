#instance(name, &block) ⇒ Object

Register an instance dependency.

An instance dependency gets created once for each instance of a dependent class.

Examples:

With a block

manufacturer.instance(:sha1_hasher) { Digest::SHA1.new }

With a Proc

manufacturer.instance(:sha1_hasher, &-> { Digest::SHA1.new })

Parameters:

  • name (Symbol, #to_sym) —

    the dependency name which will be used to access it

  • block (Block, Proc) —

    a factory method to build the dependency

Raises:

  • (Errors::RequiresCallable) —

    if no block is provided

  • (Errors::InvalidName) —

    if the name is not a symbol, cannot be converted to a symbol, or is not a valid attribute name

  • (Errors::AlreadyRegistered) —

    if the name has been registered previously
