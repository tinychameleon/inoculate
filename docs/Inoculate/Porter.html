Overview

Delivers requested dependencies to user classes.

The default method inoculate_with is named to avoid collisions with other libraries. You can alias it to something that makes your codebase legible.

Examples:

Getting Dependencies

class HealthChecker
  include Inoculate::Porter
  inoculate_with :http_service

  def healthy?
    http_service.get("/health").ok?
  end
end

With Aliasing Include

class Base
  include Inoculate::Porter[:depend_on]
end

class Child < Base
  depend_on :http_service
end

def self.[](method_name)
  m = Module.new do
    define_method(method_name) do |*names|
      names.each do |name|
        include(Inoculate.factory.build(name))
      end
    end
  end

  inclusion = Module.new
  inclusion.singleton_class.define_method(:included) do |base|
      base.extend m
  end
  inclusion
end
